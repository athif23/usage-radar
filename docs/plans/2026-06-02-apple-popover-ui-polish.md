# Apple Popover UI Polish

## Goal

Refresh the compact panel so it feels closer to the CodexBar / Apple SwiftUI popover references while staying tray-first and lightweight.

## Scope

- Keep tabs as the first surface in the panel.
- Remove unnecessary Home title chrome.
- Replace heavy home cards with native-feeling provider list rows.
- Keep provider detail pages focused on the active provider only.
- Make settings look like compact preference rows instead of stacked dashboard cards.
- Use progress bars as percent used, while text can still say percent left where appropriate.
- Keep existing provider/settings behavior intact.

## Non-goals

- No cost/account management surfaces.
- No new provider support.
- No broad layout rewrite outside the current panel UI.
- No release/version work.
